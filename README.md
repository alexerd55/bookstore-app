# Bookstore Application (Python, Tkinter)

A simple desktop bookstore app developed in Python using the Tkinter library for GUI.  
The app allows users to browse and search books, filter by genre, add/remove items
from a cart, and calculate total prices dynamically.

## Features
- Browse and search books in real time  
- Filter by genre (Programming, Self-help, Novels, etc.)  
- Add and remove items from the cart  
- Display total price and quantity dynamically  
- Simple and responsive UI built with Tkinter and Pillow  

## Technologies
- **Language:** Python 3.11  
- **GUI:** Tkinter  
- **Images:** Pillow (PIL)  
- **Editor:** Visual Studio Code  

## Future Improvements
- Add database support (SQLite or MySQL)  
- Implement login system and payment API  
- Export invoices to PDF  
- Dark mode and multi-language support
